# Checkbox

Renders a styled checkbox. Pass it the props you would pass to a standard
`<input type="checkbox">` tag.

### Usage

```js
import { Checkbox } from 'components/misc';

<Checkbox intent="success">Remember Me</Checkbox>;
```
